# [作業2]

請上 Kaggle, 在 Competitions 或 Dataset 中找一組競賽或資料並寫下：

1. 你選的這組資料為何重要
    >> 藉由過往的資料，來預測地區的犯罪行為，以便地區警局做有效的資源分配。

2. 資料從何而來 (tips: 譬如提供者是誰、以什麼方式蒐集)
    >> 由Analyze Boston 所提供，其來源為BPD官員在2015/6/14 ~ 2018/9/3 間Boston地區的犯罪報告紀錄

3. 蒐集而來的資料型態為何
    >> 結構化資料：時間、地點(經緯度)
    >> 非結構化資料：人名、地址、事件分類、事件描述。

4. 這組資料想解決的問題如何評估
    >> 分類型問題：哪類型的犯罪最常見？哪種不同類型的犯罪最有可能發生？
    >> 回歸行問題：犯罪頻率是否在一天內發生變化？週？年？
